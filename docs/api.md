## 공통 응답 스키마

### ScanResult

모든 판별 API의 공통 응답 형식입니다.

```json
{
  "verdict": "플라스틱",
  "condition": "세척 필요",
  "action": "기름기를 물로 3회 헹군 뒤 라벨을 제거하고 플라스틱 수거함에 배출하세요.",
  "reason": "PP 재질 용기이나 표면에 기름 오염이 확인됩니다."
}
```

| 필드 | 타입 | 설명 |
|------|------|------|
| `verdict` | string | 판정 결과 (아래 목록 참고) |
| `condition` | string \| null | 처리 조건. 없으면 `null` |
| `action` | string | 사용자에게 보여줄 행동 지침 |
| `reason` | string | AI 판정 근거 |

**verdict 값**

| 값 | 배출 방법 |
|----|----------|
| `일반쓰레기` | 종량제 봉투 |
| `플라스틱` | 분리배출 |
| `종이류` | 분리배출 |
| `유리` | 분리배출 |
| `캔` | 분리배출 |
| `비닐` | 분리배출 |
| `스티로폼` | 분리배출 |
| `음식물` | 음식물 쓰레기 |
| `특수폐기물` | 주민센터·지정 수거함 |

**condition 값**

| 값 | 설명 |
|----|------|
| `세척 필요` | 물로 헹궈야 재활용 가능 |
| `라벨·테이프 제거 필요` | 부착물 제거 후 배출 |
| `부품 분리 필요` | 재질별 분리 후 각각 배출 |
| `null` | 조건 없음 |

## 엔드포인트

### GET /

서버 상태 확인

**응답 예시**

```json
{
  "status": "ok",
  "service": "green-scanner-api",
  "version": "0.1.0"
}
```

---

### POST /scan/

이미지를 전송하면 AI가 분리배출 여부를 판별합니다.

**요청**

- Content-Type: `multipart/form-data`

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `image` | file | O | 촬영한 이미지 (jpg, png) |

**요청 예시 (curl)**

```bash
curl -X POST http://localhost:8000/scan/ \
  -F "image=@photo.jpg"
```

**응답 예시**

```json
{
  "verdict": "플라스틱",
  "condition": "세척 필요",
  "action": "기름기를 물로 3회 헹군 뒤 라벨을 제거하고 플라스틱 수거함에 배출하세요.",
  "reason": "PP 재질 배달 용기로 보이며 표면에 기름 오염이 확인됩니다."
}
```

---

### POST /chat/

텍스트로 물품 상태를 설명하면 AI가 분리배출 방법을 안내합니다.

**요청**

- Content-Type: `application/json`

| 필드 | 타입 | 필수 | 설명 |
|------|------|------|------|
| `message` | string | O | 물품 상태 설명 |

**요청 예시**

```json
{
  "message": "마라탕 먹고 남은 플라스틱 용기인데 기름이 많이 묻었어"
}
```

**응답 예시**

```json
{
  "verdict": "일반쓰레기",
  "condition": null,
  "action": "기름 오염이 심해 재활용이 어렵습니다. 종량제 봉투에 넣어 배출하세요.",
  "reason": "마라탕 기름은 세척이 어려운 수준의 오염으로 재활용 선별 기준을 통과하기 어렵습니다."
}
```

## 에러 응답

```json
{
  "detail": "이미지를 분석할 수 없습니다."
}
```

| 상태코드 | 설명 |
|---------|------|
| `400` | 잘못된 요청 (이미지 없음, 형식 오류 등) |
| `500` | 서버 오류 (Gemini API 호출 실패 등) |
